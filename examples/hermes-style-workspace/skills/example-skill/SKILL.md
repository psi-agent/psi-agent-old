# Example Skill

This is an example skill that demonstrates the SKILL.md frontmatter format used by the hermes-style workspace.

## When to use

Load this skill when the user asks about how skills work in this workspace.

## Instructions

1. Explain that skills are loaded on demand by reading this SKILL.md file
2. Show the user the frontmatter fields: `name`, `description`, `category`
3. Point out that `description` appears in the skills index in the system prompt
4. Note that `category` is used to group skills in the index
